# 🎬 Netflix Movies & TV Shows Analysis | Python EDA + Power BI Dashboard

## 📌 Project Overview

This project combines **Python-based Exploratory Data Analysis (EDA)** with an interactive **Power BI dashboard** to analyze Netflix’s global Movies and TV Shows catalog.

The goal was to transform raw Netflix data into meaningful visual insights by:

* Cleaning and preprocessing data
* Performing detailed exploratory analysis
* Identifying trends and patterns
* Building a professional dashboard for storytelling and business intelligence

---

## 🎯 Project Objectives

* Explore Netflix content distribution
* Compare Movies vs TV Shows
* Analyze ratings and audience categories
* Identify top genres
* Examine country-wise content production
* Study release trends over time
* Create a visually compelling dashboard for strategic insights

---

## 📂 Dataset Source

**Kaggle:** Netflix Movies and TV Shows Dataset

# 🔍 Key Insights

### 🎥 Content Type:

* Movies dominate Netflix’s library
* TV Shows have shown strong recent growth

---

### 🌍 Geographic Trends:

* United States leads content production
* India is the second-largest contributor
* Global diversification is increasing

---

### ⭐ Audience Focus:

* TV-MA and TV-14 dominate
* Netflix heavily targets mature viewers

---

### 📅 Growth:

* Significant expansion after 2015
* Peak release volume around 2019–2020

---

### 🎭 Genres:

* Drama and International Movies dominate
* Documentaries and Comedy are major segments
